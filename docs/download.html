Compiling From Source

  1. Check out the commit you wish to build in Git, or download and extract a ZIP/tarball of the latest development version from GitHub.
  2. cd into the source directory. Call cmake . and then make.
  3. You need to have CMake and Make installed in order to do run these commands. CMake will check that all required libraries are installed; if this fails, install the libraries it complains about, and retry until all dependencies are found.
  4. You can run FreeRCT via ./bin/freerct or make run.
